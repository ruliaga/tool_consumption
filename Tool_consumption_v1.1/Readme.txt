1. Сохраните запрос по заявкам из 1С в папке с программой в формате xlsx
2. Из файла с заявками удалите скрытые строки и преобразуйте диапазон в таблицу
3. Запустите файл tool_consumption.exe
4. Дождитесь окончания работы программы. Должно появиться сообщение о количестве обработанных файлов.
5. Программа генерирует файл tool_consumption.xlsx, в котором представлен расчет суммарного расхода инструмента.
6. Также программа создает файл Discription.txt, в котором приводится расшифровка расчета. Расшифровку можно использовать
для проверки корректной работы программы.
