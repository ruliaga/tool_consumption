-------------------------
  Tool_consumption_v1.0
-------------------------
1. Программа работает только с форматом xlsx. Файлы формата xls необходимо пересохранить в xlsx.
2. Файлы КН, которые планируется проверить, необходимо скинуть в отдельную папку.
3. Файлы КН должны быть закрыты.
4. В папку с КН необходимо скопировать Tool_consumption_v1.0.exe.
5. Запускаем Tool_consumption_v1.0 от имени администратора.
6. После запуска программы в рабочей папке генерируется файл "Суммарный расход инструмента.xlsx", в котором отображается результат расчета.
7. Для повторного запуска программы в данной папке необходимо удалить файлы, сгенерированные во время предыдущего запуска: файл "Суммарный расход инструмента.xlsx" и файл "Ошибки_Расход инстр. На 1-ну дет.txt".

В связи с тем, что в КН существуют некоторые различия в оформлении, могут появиться ошибки двух типов:
1. Фатальные ошибки - ошибки которые приводят к остановке работы программы. При этом на экран выводится сообщение с именем проблемного файла. Для продолжения работы необходимо привести файл к общепринятому оформлению или удалить этот файл из рабочей папки.
2. Ошибки заполнения поля "Расход инстр. На 1-ну дет" при отсутсвии значения в ячейке. При этом программа выполняется до конца и в рабочей папке генерируется файл "Ошибки_Расход инстр. На 1-ну дет.txt", в котором указаны файлы с данной ошибкой. Необходимо заполнить отсутствующие значения, чтобы они учитывались в расчете.
